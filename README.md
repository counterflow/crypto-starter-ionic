## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

* Node.js and npm

### Installing

```
npm install -g ionic
```

```
cd apps/angular
npm i
ionic serve

cd apps/react
npm i
ionic serve

cd apps/vue
npm i
ionic serve
```
